## サンプルコードについて
本リポジトリではオライリー公開しているサンプルコードを使用しています。
各ノートブックを実行する際には以下のコマンドを使用してください。
```
git clone https://github.com/oreilly-japan/deep-learning-from-scratch-2.git
```
コマンド実行後、ディレクトリ名を**samplecode**に変更してください。
